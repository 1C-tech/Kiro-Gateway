API Key 管理

Admin API Key
-
用于后台管理面板的所有请求（账号管理、导入导出、系统设置）。请求头格式：Authorization: Bearer <key>
Proxy API Key
-
用于外部客户端（Cursor、VS Code 等）访问 Kiro API 代理。启动网关时通过 PROXY_API_KEY 环境变量设置。

代理设置

设置 Kiro API 的出站代理（HTTP/SOCKS）
